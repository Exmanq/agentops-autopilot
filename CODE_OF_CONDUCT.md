# Code of Conduct

We are committed to a welcoming and inclusive community.

## Our Standards
- Be respectful and considerate.
- Give and gracefully accept constructive feedback.
- Focus on what is best for the project and community.
- Show empathy towards other community members.

## Unacceptable Behavior
- Harassment, intimidation, or personal attacks.
- Discriminatory language or imagery.
- Trolling or insulting comments.

## Responsibilities
- Project maintainers enforce this Code of Conduct.
- Maintainers may remove or reject contributions not aligned with this code.

## Reporting
Report incidents to the maintainers at `security@agentops.local`. Reports will be reviewed promptly and confidentially.

## Scope
This Code applies within project spaces and in public spaces when representing the project.
